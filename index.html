🚀 Getting Started

  1. Set up Supabase: Create a free account at supabase.com
  2. Run the SQL: Copy the database schema and run it in your Supabase SQL editor
  3. Update credentials: Replace YOUR_SUPABASE_URL and YOUR_SUPABASE_ANON_KEY in the code
  4. Deploy: Upload to Vercel for free hosting

  • ✅ Multi-device access
  • ✅ Automatic cloud backups
  • ✅ User authentication
